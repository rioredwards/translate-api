About

This is a Simplified version of the Google Translate API for students at Code The Dream.

It's technically a Proxy that forwards your requests to their server, so you don't have to deal with Google's Authentication system.

You're currently viewing the default response for a GET request to the home route of my API. My server responds with an HTML page and your browser renders it to the screen.

Usage:

Because this API is supposed to translate some text that you send over. You will need to send a POST request to this same URL. POST requests allow you to send your own data to the API.

Please send a POST request that includes:

  • some text to translate
  • the target language

Your fetch should look like this:

fetch(https://translate-424302.uc.r.appspot.com/, {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    text: INSERT_TEXT_HERE,
    language: INSERT_LANGUAGE_HERE,
  }),
});

The text may be a string or an array of strings
